## 2026-04-28 — My Farm pivot + auth pivot

A coordinated pivot covering auth model, landing surface, My Farm headline, navigation, and a new RPC.

- **Auth model flipped to public-by-default.** Middleware now denylists only `/my-farm`; everything else (Overview, Grain detail, US, Seeding) is publicly accessible. Root route redirects to `/overview`.
- **Landing page retired.** `app/page.tsx` reduced to a one-line redirect; `components/landing/` and `app/api/trial-notify/route.ts` deleted. Bio-trial design / handover docs marked deprecated.
- **My Farm headline changed.** New `grain-storage-card.tsx` (two-input total + remaining tonnes per grain, with the "X% of farmers have more in the bin than you" peer comparison) is the prominent slot. The cross-grain sentiment voting block is removed from the page; components paused not deleted.
- **Nav reorder shipped.** `My Farm` tab moved out of the central pill and anchored to the right cluster of the header (`components/layout/my-farm-nav-link.tsx`). Mobile sheet reordered so My Farm sits at the bottom of the primary nav block.
- **New RPC.** `get_grain_storage_comparison(p_grain)` ships peer comparison with a ≥5-farmer privacy gate.

Components retired in this pivot (paused, files retained):

- `multi-grain-sentiment.tsx` — cross-grain Holding/Hauling voting card on `/my-farm`. Removed from page render in favor of the new `grain-storage-card.tsx`.
- `sentiment-banner.tsx` — cross-grain sentiment overview rail. Paired with `multi-grain-sentiment` retirement.

**Restoration path (if ever promoted to an active feature track):** re-import the components in `app/(dashboard)/my-farm/page.tsx`, re-add the `getSentimentOverview` + `getUserSentimentVote` parallel fetches, and put the section back between *Weekly Summary* and *Grain in your bin*. The server action `voteSentiment()` and the `grain_sentiment_votes` / `sentiment_history` / `sentiment_daily_rollup` tables are all retained.
